# Security Services

Secret management, encryption, access control, and security scanning.

## Future Integrations

- HashiCorp Vault for secret management
- SOPS for secrets operations and encryption
- Falco for runtime security monitoring
- OPA/Gatekeeper for policy enforcement
- Sealed Secrets for Kubernetes secret encryption

## Structure

Each security service will follow the standard chart structure extending `platform/base-chart`.

See [../../platform/base-chart/README.md](../../platform/base-chart/README.md) for chart guidelines.
