# Storage Services

Data persistence, backups, archives, and distributed storage.

## Future Integrations

- MinIO for S3-compatible object storage
- PostgreSQL for relational data
- MongoDB for document data
- Redis for caching
- Backup solutions (Velero, Restic)
- Archive and long-term storage

## Structure

Each storage service will follow the standard chart structure extending `platform/base-chart`.

See [../../platform/base-chart/README.md](../../platform/base-chart/README.md) for chart guidelines.
